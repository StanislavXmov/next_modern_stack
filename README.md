# next_modern_stack

To install dependencies:

```bash
bun install
```

To run the web app:

```bash
bun run dev:web
```

If port 3000 stays occupied after a failed shutdown (Windows), run:

```bash
bun run dev:web:kill
```

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
